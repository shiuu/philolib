# PhiloLib
A library full of philosophy books.

## To run the library
Run the following commands:
```shell script
git clone https://github.com/shiuu/philolib.git
cd philolib
mvnw spring-boot:run
```
Or `mvn spring-boot:run`

Then browse to: 
http://localhost:8088/
